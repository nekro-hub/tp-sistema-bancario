# Sistema de Gestión Bancaria

Trabajo práctico integrador desarrollado en Python.

## Funcionalidades

- Alta de cuentas
- Baja de cuentas
- Modificación de datos
- Depósitos y extracciones
- Consulta de cuentas
- Listado de clientes

## Tecnologías

- Python 3
- Git
- GitHub

## Objetivo

Simular el backend de un sistema de gestión bancaria utilizando estructuras de datos y funciones en Python.
